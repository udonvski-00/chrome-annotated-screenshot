## 機能
- ページ内の画像や動画にファイル名 / URL のラベルを付けた状態でスクリーンショットを保存。
- スクリーンショット（JPG）と同時に URL 一覧の TXT を出力。
- 追加の設定なしでそのまま利用可能。

![スクリーンショット 2025-09-22 085128](https://github.com/user-attachments/assets/2d26bfd6-0546-436a-9a31-ef127932494a)

## キャンセル
- 撮影中に `Esc` を押すと、その時点までに撮ったフレームを使って保存します。
- モード選択ダイアログ表示中に `Esc` を押した場合は何も保存せずに閉じます（0.3.0 から）。

## 制約事項
- `chrome://` や `chrome-devtools://` などの内部ページでは動作しません。
- 無限スクロールなどページ構造によっては最後まで撮れない場合があります。
- 背景画像はサイトの実装によって取得できないことがあります。
- OS やブラウザの違いにより、まれに余白が入ることがあります。

## フィルタリング
- `data:` / `blob:` URL はスキップ。
- 1x1 GIF などのトラッキング画像は除外。
- 設定で除外ドメインやパターンを追加可能。

## 利用している権限
- `tabs`, `activeTab`, `scripting`：ページへのスクリプト注入やスクロール制御。
- `downloads`：生成した JPG / TXT を保存するために使用。

## バージョン 0.3.0 のアップデート

- モード選択ダイアログを刷新し、ボタンを「ページ全体（画像＋テキスト）」と「選択範囲スクロール」の 2 つに整理。
- ダイアログ表示中に `Esc` を押すと何も保存せず中断するように変更。
- 選択範囲スクロールの合成処理を調整し、下端が欠けるケースを解消。
- モーダルや進捗表示の日本語表記を整備し、文字化けを修正。
- バージョン番号を `0.3.0` に更新。
